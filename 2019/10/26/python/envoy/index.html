python envoy 源码解析

envoy

envoy 库的介绍上说它是一个可以被人类理解的 python 子进程。可以看出这个库的主要做用是使得 python 子进程的使用更加便利高效，并且符合人类的使用习惯。

前置知识点

看源码前需要对下面几个 python 原生库的使用有了解：

  1. shlex
  2. subprocess.Popen
  3. threading.Thread

可以参考本文最后的参考资料。我也是边看源码边看查阅资料 🙂 ，感觉这样比较高效
